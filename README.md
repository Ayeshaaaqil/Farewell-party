# ⚜️ A Tribute to Dr. Ayesha - Farewell Website

An elegant, premium, and interactive digital farewell tribute website dedicated to our exceptional Batch Incharge, **Dr. Ayesha**. Built collectively on behalf of the entire batch to express our deepest gratitude, respect, and admiration for her untiring guidance and mentorship.

---

## 🎨 Visual Aesthetics & UI Features

- **Royal Velvet Theme:** A premium combination of ultra-dark royal purple (`#0d0518`) and radiant liquid gold accents mimicking a luxury award ceremony.
- **Glassmorphism UI:** Modern frosted-glass layouts with clean borders and translucent background reflections.
- **Experiential Landing Page:** A clean, minimal entry page featuring a classic typography reveal layout.
- **Interactive Celebration Engine:** Clicking the **"Enter Experience"** button triggers a dynamic CSS/JS burst of balloons and celebratory emojis flying up the viewport.
- **Ambient Gold Particles:** A custom lightweight HTML5 Canvas script that generates drifting, glowing golden micro-particles in the background.
- **Fully Responsive Architecture:** Optimized beautifully for seamless rendering on smartphones, tablets, and high-resolution desktop viewports.
